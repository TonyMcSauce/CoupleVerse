# CoupleVerse 💝

**Play, laugh, and discover each other again.**

A premium collection of couple games — beautiful, browser-native, zero backend needed.

## 🎮 Games Included

1. **Guess My Answer** — Answer secretly, reveal together, see if your hearts align
2. **Deep Questions** — A deck of meaningful relationship questions
3. **Truth or Dare (Couple Edition)** — Sweet, playful, and a little spicy
4. **Compliment Battle** — Alternate compliments with a timer for pressure
5. **Memory Lane** — Test how well you remember your story together
6. **Emoji Story** — Tell relationship memories in emojis only
7. **Date Idea Generator** — A spinning wheel of date night ideas

## 🚀 Deploy to GitHub Pages

1. Fork or clone this repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. Visit `https://yourusername.github.io/coupleverse`

## 🛠 Tech

- Pure HTML, CSS, JavaScript — no framework, no build step
- Offline-capable after first load
- Mobile-first responsive design
- Dark mode / Light mode toggle
- Web Audio API for sound effects
- localStorage for game progress

## 📁 Structure

```
coupleverse/
├── index.html     # App shell + all pages
├── style.css      # All styles + animations
├── script.js      # All games + logic
└── README.md
```

## ✨ Features

- Apple-inspired glassmorphism design
- Playfair Display + DM Sans typography
- Smooth animations and transitions
- Ambient blob background with floating particles
- Sound effects via Web Audio API
- Full dark mode support
- 7 fully playable games with rules, scores, and state
